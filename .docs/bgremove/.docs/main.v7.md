## 🏗️ 각 파트별 괴물들 라인업

### 1. 웹에서 AI 돌릴 판을 깐 괴물: 마이크로소프트 (MS)

JS에서 사용하는 AI 실행 엔진의 표준 격인 `ONNX Runtime`을 주도해서 만든 곳이 바로 **마이크로소프트**야.
원래는 서버나 PC에서 AI를 빠르게 돌리려고 만든 엔진인데, 얘네가 "이걸 웹 브라우저(WebAssembly/WebGL)에서도 돌아가게 만들면 대박이겠는데?" 하고 웹 버전(`ONNX Runtime Web`)을 오픈소스로 풀었어. 브라우저가 그래픽카드를 직접 쥐어짜서 연산하게 만든 일등 공신이야.

### 2. 브라우저 엔진(V8)과 WebGPU를 다진 괴물: 구글 (Google)

자바스크립트가 실행되는 **V8 엔진**을 만들고 발전시킨 곳은 **구글**이야. 구글은 V8을 미친 듯이 최적화해 둔 것도 모자라, 최근에는 아예 브라우저가 유저의 최신 그래픽카드(GPU) 자원을 네이티브 수준으로 100% 다 쓸 수 있게 만드는 **WebGPU 표준 사양**을 W3C와 함께 주도해서 정착시켰어. 판은 MS가 짰지만, 그 밑바닥 하이웨이를 뚫어준 건 구글과 브라우저 연합군인 셈이지.

### 3. 배경 제거 AI 모델을 깎아 만든 괴물: 오픈소스 커뮤니티 (U-2-Net 등)

"그럼 이미지에서 배경만 기가 막히게 분리해내는 AI 알고리즘은 누가 만들었냐?"
이건 수많은 인공지능 학자와 대학 연구원들이 논문으로 발표한 **오픈소스 컴퓨터 비전 모델**들이 기반이야. (예를 들면 `U-2-Net`, `TRACER` 같은 논문 기반 모델들).
중국 친황다오 대학 연구팀이나 글로벌 AI 커뮤니티에서 "배경만 칼같이 따는 가벼운 신경망 구조를 알아냈다!" 하고 모델을 공개하면, 전 세계 형님(?)들이 그걸 가져다가 웹용(ONNX)으로 뼈와 살을 깎아 다이어트를 시킨 거지.

### 4. 이걸 한 줄짜리 라이브러리로 포장한 괴물: @imgly (기업)

마지막으로 네가 코드에서 `imglyRemoveBackground()` 한 줄 띡 호출하면 돌아가게끔, [AI 모델 로드 + ONNX 엔진 가동 + 웹 워커 분리 + 결과물 Blob 변환]이라는 개판(?) 오분 전의 복잡한 과정을 예쁘게 캡슐화해서 패키지로 던져준 곳은 `IMG.LY`라는 디지털 이미지 처리 전문 글로벌 기업이야.
